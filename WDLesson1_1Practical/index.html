Web Technologies

HTML

A html also called HyperText Markup language is the standard for making web pages. Html also describes the structure of the web page and consists of multiple elements. It tells the browser how to display the content as well. This is the foundational language for structing content. It's a system of "tags" to define the overall structure of the content in the web page.

CSS

A CSS is a language we use to describe a web page. It dictates how html elements are dsplayed on multiple medias like screens and more. It can also define the style and overall layouts of web pages for visual appeal.

Java Script

A Javascript is computer programming language that is commonly used to create interactive effects inside of a web browser. It's used to create websites that you the user can engage with and so it has applications for interaction.
